Updating Visual Studio 2012 and NuGet Package Manager

Updates for Visual Studio and its extensions are managed in the Extensions and Updates dialog.

NuGet Package Manager should already be installed with Visual Studio 2012, but it might not be up to date. As of October 2013, the newest version is 2.7.

On the left, navigate to Updates. If there are any updates to install, make sure to do so before continuing with the Vidview SDK Getting Started Guide.

If you are running Visual Studio on a system that is not connected to the Internet, you can download the latest version of NuGet Package Manager as an installer from the Visual Studio Extension Gallery.
