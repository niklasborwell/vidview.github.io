Example of automatic sharp stills selector

Background

This shows the Imint roadmap feature of automatically select the sharpest video frame to use as snapshot. Imint Vidhance meta analysis already provide information on video frame sharpness, which would be utilized in this function. On a moving platform, some video frames will be sharper than others, due to vibrations. The feature will work like this: when an operator clicks to select a snapshot from moving video, the function will search in a narrow band of previous and future frames , selecting as a snapshot the one with best visibility. Increasing the band will provide better sharpness, while a more narrow search band will ensure timing accuracy.

Example

Below are five consecutive frames from an aerial video from a helicopter platform. In this example the search band is set to 2, which means that the current frame and +/- two frames will be analysed, in total 5 frames.

  1. Frame n-2
  2. Frame n-1
  3. Frame n
  4. Frame n+1
  5. Frame n+2

In this example, the frame n+1 would be automatically selected to be used as snapshot still image.
