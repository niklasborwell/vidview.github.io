Vidhance Track

Introduction

This document gives some additional details on the Vidhance real-time object tracker, and its access from Vidview.

Vidhance Tracking

The Vidhance object tracking is designed to follow objects over consecutive frames. It can automatically determine the correct size and location for moving objects. The object tracker can handle several difficulties associated with following objects such as

Occlusion
If the objects partially or completely disappears for a short period.
Frames out of focus
If the camera loses focus.
Shape changes
For instance if the objects turns around.
Light and color changes
For instance if the object passes through a shadowed area.

Modes and adaptability in Vidview

The object tracker has two different modes of operation.

Static
The tracking box size will be constant. Useful if the object stays at a constant distance from the camera and doesn't change shape.
Adaptive
The tracker will automatically calculate the size of the object being tracked and adapt the tracking box size accordingly. Useful if the object moves back and forth from the camera or change shape.

There are several commands and parameters, accessible through the SDK.

active
Shows whether the tracker is currently tracking or not.
bounds
The bounding box of the object being tracked.
quality
The quality of the current tracking. A lower value could indicate that the object is difficult to follow.
complexity
Parameter to control how well and for how long the tracker can handle occlusion.
defaultsize
The default bounding box start size.
enabled
Enables or disables using the mouse pointer to start tracking. The left mouse button can be used to start tracking at a certain place using the standard size. The right mouse button can be used to drag a bounding box of desired size for tracking.
maximumsize
The largest allowed size for the bounding box when using adaptive mode.
minimumsize
The smallest allowed size for the bounding box when using adaptive mode.
mode
The tracking operation mode, either static or adaptive.
start
Starts tracking.
stop
Stop tracking.
