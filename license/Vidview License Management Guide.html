Vidview License Management Guide

Introduction

Vidview in itself does not require a valid license. A valid license is only required when using Vidhance® video enhancements or to use codec packs bought through Imint.

A license key is required to create a new license.

Simple Activation

The easiest way to activate is to connect your computer to the Internet and then start Vidview Window with the following commandline: Imint.Vidview.Window.exe -s "license.request 01234ABCDabcd" where 01234ABCDabcd is replaced with your license key.

Next time Vidview is started the new license will be used.

Advanced Activation

In order to have more control over the activation process it is possible to activate the Vidview settings console. It is activated by starting the following command line: Imint.Vidview.Window.exe -r console://. When started this way the Vidivew settings console apear.

To verify if there currently is a valid license loaded type: license.valid and press enter. It will return license.valid true if the license is valid.

To request a license type: license.request 01234ABCDabcd where 01234ABCDabcd is replaced with your license key. Please note that the computer has to be connected to the Internet to request a license this way.

Offline Activation

Offline activation is used when the computer running Vidview can't be connected to the Internet for security reasons. Start the offline activation process by enabling the Vidview settings console by starting the command line: Imint.Vidview.Window.exe -r console://

When the settings console appears type: license.address 01234ABCDabcd where 01234ABCDabcd is replaced with your license key. When pressing enter a URL is returned looking something like: https://aurora.imint.se/license/01234ABCDabcd/abcdABCD0123456789. Type that URL into the address field of an Internet connected computer and press enter. It will then open a XML-file. Save that file the file and transfer it to the computer running Vidview. In the settings console type: license.load file:///e:/license.xml. Replace the path to fit the current situation. After pressing enter and restarting Vidview the license will be in use.

Deactivation

To deactivate your license, go to your ProgramData\Imint folder, typically C:\ProgramData\Imint, and remove the file license.xml
